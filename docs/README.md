## GitHub Pages setup

1. Push this repository to GitHub.
2. In GitHub: **Settings â†’ Pages**
3. Set **Source** = "Deploy from a branch"
4. Set **Branch** = `main` (or your default branch)
5. Set **Folder** = `/docs`

Then use:
- `https://<your-username>.github.io/<repo-name>/privacy`
- `https://<your-username>.github.io/<repo-name>/terms`

Replace the `REPLACE_WITH_YOUR_SUPPORT_EMAIL` placeholders in `docs/privacy.md` and `docs/terms.md` before submitting to any platform for review.
